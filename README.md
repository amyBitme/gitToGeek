可以使用Visual Studio Code觀察
## 創建 
創立資料夾
```
mkdir gitToGeek
```

進入資料夾
```
cd gitToGeek
```

啟動git
```
<!-- 資料夾內會新增 .git的隱藏資料夾-->
git init
```

將檔案新增至暫存區
```
<!-- 請注意有空格 -->
git add .
```

提交至儲存庫
```
git commit -m "這裡寫你想新增的筆記"
```

---
## 查詢
查看git的狀態
```
<!-- 檔案是否更動-->
git status
```
查看提交的筆記紀錄
```
git log
```
